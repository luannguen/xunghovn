# Contributing to Xưng Hô VN

Thank you for contributing to Xưng Hô VN!

## Code Standards & Governance
- Follow 3-Layer Architecture: UI -> Hooks -> Logic/Services.
- Ensure all kinship logic rules pass validation in `src/lib/kinshipLogic.ts`.
- Run `./components/instruction/scripts/validate.ps1` before submitting Pull Requests.
